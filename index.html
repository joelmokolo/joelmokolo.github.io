Priority Project — Maternity Focus

NHS Maternity Outcomes Analysis — MSDS Data

SQL  •  Excel  •  NHS Open Data  •  MSDS  •  Data Quality

🔨 In build — pre-interview priority

End-to-end analysis of NHS England maternity outcomes using the Maternity Services Data Set (MSDS) — the same national dataset Badgernet feeds into at NUTH. SQL extraction and cleaning, Excel pivot analysis, trend identification across delivery types, gestation outcomes and booking rates by trust.

Why this project matters

The MSDS is the national reporting spine for maternity services. Understanding its structure, gaps and anomalies is exactly the work of a Badgernet information analyst. This project demonstrates that knowledge before interview.

Analysis scope

  • Booking within 10 weeks — completeness analysis
  • Delivery type trends (SVD, CS, instrumental)
  • Gestation outcomes and preterm rates
  • Identifying missing records and validation flags

Example SQL — Booking Completeness

SELECT
    trust_code,
    COUNT(*) AS total_bookings,
    SUM(CASE WHEN booking_weeks <= 10 THEN 1 ELSE 0 END) AS booked_early,
    ROUND(
        SUM(CASE WHEN booking_weeks <= 10 THEN 1 ELSE 0 END) * 100.0
        / COUNT(*), 1
    ) AS pct_booked_early,
    SUM(CASE WHEN booking_weeks IS NULL THEN 1 ELSE 0 END) AS missing_booking_date
FROM maternity_bookings
GROUP BY trust_code
ORDER BY pct_booked_early ASC;

Key analytical question

Which trusts show the highest rate of missing or late booking data, and what does that mean for MSDS submission quality and national benchmarking?

Clinic DNA Rate Analysis & Digital Improvement Case

  • SHINE sustainability metrics embedded
  • Mirrors real improvement analyst work at NUTH
  • Demonstrates data-to-recommendation capability

Example SQL — DNA Rate by Wait Band

SELECT
    CASE
        WHEN days_wait BETWEEN 0  AND 7  THEN '0-7 days'
        WHEN days_wait BETWEEN 8  AND 14 THEN '8-14 days'
        WHEN days_wait BETWEEN 15 AND 21 THEN '15-21 days'
        ELSE '22+ days'
    END AS wait_band,
    COUNT(*)                                                      AS total_appts,
    SUM(CASE WHEN attendance = 'DNA' THEN 1 ELSE 0 END)          AS dna_count,
    ROUND(
        SUM(CASE WHEN attendance = 'DNA' THEN 1 ELSE 0 END)
        * 100.0 / COUNT(*), 1)                                   AS dna_pct
FROM appointments
GROUP BY
    CASE
        WHEN days_wait BETWEEN 0  AND 7  THEN '0-7 days'
        WHEN days_wait BETWEEN 8  AND 14 THEN '8-14 days'
        WHEN days_wait BETWEEN 15 AND 21 THEN '15-21 days'
        ELSE '22+ days'
    END
ORDER BY dna_pct DESC;
